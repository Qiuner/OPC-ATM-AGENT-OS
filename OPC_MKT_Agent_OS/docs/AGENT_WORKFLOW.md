# 多 Agent 工作流（v0.1）

## Agent 角色

1. Strategist Agent（策略官）
- 输入：品牌定位、产品、目标人群、阶段目标
- 输出：内容策略、渠道分配、周计划

2. Content Architect Agent（内容架构师）
- 输入：策略计划
- 输出：选题池、内容大纲、内容漏斗（认知/信任/转化）

3. Copy/Script Agent（文案脚本官）
- 输入：选题与上下文素材
- 输出：小红书文案、抖音口播脚本、公众号长文、推特线程

4. Publisher Agent（分发官）
- 输入：内容包、发布时间规则
- 输出：发布队列、平台格式化内容

5. Analyst Agent（复盘官）
- 输入：平台数据
- 输出：高表现题材、低表现原因、下轮优化建议

## 控制规则
- 人工审核后才能发布（human-in-the-loop）
- 平台敏感词与规则先过滤
- 每次迭代只改 1-2 个变量（标题/封面/CTA）
