# OPC_MKT_Agent_OS - MVP 开发方案（14天）

## 阶段 1：资产模型（D1-D3）
- 定义用户上下文 schema
- 完成 `data/context.example.json`
- 定义平台内容模板（XHS/抖音/视频号/X/即刻）

## 阶段 2：多 Agent 流程（D4-D8）
- Planner Agent：产出主题与节奏
- Content Agent：产出各平台草稿
- Publisher Agent：产出发布日历与待发布包

## 阶段 3：测试与迭代（D9-D14）
- 选 1 个样板账号（你自己的）连续发布 7 天
- 采集互动指标（曝光/收藏/私信咨询）
- 调整标题模板、封面钩子、CTA

## MVP 验收标准
- 能从上下文一键生成 7 天内容计划
- 能导出 5 平台发布草稿
- 每条内容含 CTA 与引流路径
- 形成复盘报告（哪些内容值得加倍投放）
