## 二、Demo 故事线设计（3分钟演示视频核心）

### 故事：「一个出海品牌如何用 AI 营销团队在 10 分钟内完成一周的营销工作」

**完整流程（端到端）：**

```
用户输入一句话需求（如 "为我们的新品做一波全渠道推广"）
     ↓
CEO Agent 自动拆解任务 → 生成营销计划（Plan）
     ↓
用户审批计划（人机协同 — 一键审批/修改）
     ↓
多 Agent 并行执行：
  ├── X/Twitter Agent → 生成推文系列
  ├── Email Agent → 生成邮件营销内容
  ├── XHS Agent → 生成小红书笔记
  ├── Blog/SEO Agent → 生成博客文章
  └── Brand Compliance Agent → 自动审核所有内容
     ↓
内容进入审批流水线（AI 审核 + 人工确认）
     ↓
一键发布到多平台 → 数据回流 → 分析优化
```

## 五、得分最大化策略

### 🎯 商业潜力（25%）— Jayden 主攻
- **TAM 数据**：全球数字营销市场 $786B (2026)，出海企业营销自动化渗透率 < 5%
- **商业模式**：SaaS 订阅制（基础版 $99/月 → 企业版 $999/月）
- **差异化**：不是 ChatGPT wrapper，是**多 Agent 协作 + 人机协同审批 + 数据飞轮**
- **落地路径**：先做 DTC 品牌出海 → 扩展到本地化营销 → 企业级营销中台

### 🎯 问题定义与创新性（20%）— Jayden 主攻
- **痛点锐化**：一个出海品牌要运营 5+ 平台，需要 5+ 内容运营人员，月成本 $15K+
- **创新点**：① 多 Agent 专业分工（非单一大模型）② 人机协同审批（非全自动黑盒）③ 数据飞轮自进化

### 🎯 Agent 能力深度（20%）— 陈总 主攻
- Demo 中要展示：Agent 之间的**对话和协作**（不是独立执行）
- CEO 要展示**思考过程**（为什么这样分配，为什么选这个角度）
- Brand Compliance 要**拒绝不合规内容**并给出修改建议（展示约束能力）

### 🎯 产品体验（20%）— 罗总 主攻
- 操作流畅无卡顿 > 功能多
- 每一步都有**视觉反馈**（loading 状态、进度条、成功提示）
- 多平台预览要**逼真**（像真的发到了 Twitter/小红书）

### 🎯 工程完整度（15%）— 邱总 主攻
- 代码在 GitHub，README 清晰
- 架构图在 PPT 中展示
- 线上 Demo 可访问
- 分工明确，协作有序
