# 版式系统（8 种标准布局）

## 按信息密度分类

### 1. sparse（留白型）
- **信息密度**: 20-30%
- **留白占比**: 70-80%
- **信息点数**: 1-2 个
- **适用**: 封面图、品牌感内容、情绪类、高级感
- **位置策略**: 核心信息集中在画面中心或黄金分割点，四周大面积留白
- **Prompt 指导**: "minimalist composition with large negative space, single centered focal point, breathing room around all elements, clean uncluttered design"

### 2. balanced（均衡型）
- **信息密度**: 40-50%
- **留白占比**: 50-60%
- **信息点数**: 3-4 个
- **适用**: 通用型，大多数笔记、图文内容、产品介绍
- **位置策略**: 上部标题区 → 中部主体内容 → 下部补充/CTA，视觉重心偏上
- **Prompt 指导**: "balanced layout with clear visual hierarchy, title area at top 20%, main content in center 60%, supporting info at bottom, even distribution of elements"

### 3. dense（密集型）
- **信息密度**: 60-75%
- **留白占比**: 25-40%
- **信息点数**: 5-8 个
- **适用**: 干货教程、多步骤指南、对比总结、知识卡片
- **位置策略**: 网格化排列，用分隔线/色块/编号区分内容块，最大化信息承载
- **Prompt 指导**: "information-dense infographic, structured grid layout, multiple content blocks separated by dividers or color zones, compact yet readable, maximum information per area"

## 按结构分类

### 4. list（列表型）
- **信息密度**: 50-60%
- **留白占比**: 40-50%
- **信息点数**: 4-8 个
- **适用**: Top N 推荐、步骤教程、清单、编号指南
- **位置策略**: 纵向依次排列，每项配编号/图标，等间距，左对齐
- **Prompt 指导**: "numbered list layout, vertical arrangement of items, each item with number or icon on left and text on right, consistent spacing between items, clear sequential reading flow"

### 5. comparison（对比型）
- **信息密度**: 55-65%
- **留白占比**: 35-45%
- **信息点数**: 2-4 组对比
- **适用**: 产品对比、before/after、优缺点、VS 对决
- **位置策略**: 左右分栏（竖版）或上下分栏，中间明确分隔线/VS 标记
- **Prompt 指导**: "split comparison layout, side by side or top bottom arrangement, clear dividing line or VS marker in center, matching structure on both sides, parallel visual treatment"

### 6. flow（流程型）
- **信息密度**: 45-55%
- **留白占比**: 45-55%
- **信息点数**: 3-6 个步骤
- **适用**: 流程说明、时间线、操作指南、故事线
- **位置策略**: 箭头/连线引导阅读方向，步骤间有明确的顺序关系
- **Prompt 指导**: "flowchart style layout, connected steps with arrows or lines, clear directional reading flow from top to bottom or left to right, numbered sequential stages"

### 7. mindmap（脑图型）
- **信息密度**: 50-65%
- **留白占比**: 35-50%
- **信息点数**: 1 中心 + 3-6 分支
- **适用**: 知识体系、概念总结、关系图、分类展示
- **位置策略**: 中心概念居中，向外辐射分支，层级递减
- **Prompt 指导**: "mind map layout, central concept in the middle radiating outward with branches, hierarchical structure from core to details, connecting lines between related concepts"

### 8. quadrant（四象限型）
- **信息密度**: 55-65%
- **留白占比**: 35-45%
- **信息点数**: 4 个区块
- **适用**: 2×2 矩阵分析、分类总结、四维度对比
- **位置策略**: 四等分，每区块独立内容，中心交叉线/标签
- **Prompt 指导**: "four quadrant matrix layout, 2x2 grid with clear axis labels, each quadrant containing distinct content, central crosshair or dividing lines"

## 版式选择速查

| 你要做什么？ | 推荐版式 |
|-------------|----------|
| 封面/第一眼吸引 | sparse |
| 一般内容展示 | balanced |
| 干货/教程/知识卡 | dense |
| Top N / 步骤列表 | list |
| A vs B / 优缺点 | comparison |
| 操作流程/时间线 | flow |
| 知识体系/概念图 | mindmap |
| 矩阵/分类/四维 | quadrant |
