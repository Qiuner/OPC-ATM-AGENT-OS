# 小红书画布规范

## 标准尺寸

| 用途 | 尺寸 | 宽高比 | generate_image aspectRatio | 推荐场景 |
|------|------|--------|---------------------------|----------|
| 竖版封面（推荐） | 1080×1440 | 3:4 | `3:4` | 信息图、教程、种草笔记 |
| 方形 | 1080×1080 | 1:1 | `1:1` | 产品展示、对比图 |
| 全屏竖版 | 1080×1920 | 9:16 | `9:16` | 视频封面、全屏海报 |

**默认推荐**: 3:4 竖版 — 小红书流量最高、互动最好的比例。

## 安全区域（Safe Zones）

```
┌──────────────────────────────┐
│                    ⚠️ 右上角  │ ← 点赞/收藏/分享按钮
│                    (80×160px) │    不要放文字
│                              │
│   ┌──────────────────────┐   │
│   │                      │   │
│   │   ✅ 有效内容区域     │   │
│   │   (安全创作区)        │   │
│   │                      │   │
│   │                      │   │
│   └──────────────────────┘   │
│                              │
│ ⚠️ 底部 10%                   │ ← 标题栏遮挡
│ 禁止放置关键信息             │    绝对不放文字
│                    ⚠️ 右下角  │ ← 水印区域
└──────────────────────────────┘
```

### 规则
1. **底部 10%**: 小红书 APP 叠加标题栏，完全遮挡底部约 10% 内容。**绝不在此放文字或关键元素。**
2. **右上角 (~80×160px)**: 点赞、收藏、分享按钮覆盖区域。**不放文字。**
3. **右下角 (~80×80px)**: 平台/用户水印区域。**不依赖此区域传达信息。**
4. **有效内容区**: 上部 85%、左侧 90% 为安全创作区。

### Prompt 安全区追加语（每个 prompt 末尾必加）
```
Leave bottom 10% of image completely empty — no text, no key visual elements.
Avoid placing any text in the top-right corner area.
```

## 系列图一致性规则

### 制作系列图（多张卡片）时
1. **封面图先生成** — 第 1 张确定整体风格基调
2. **后续图使用 --ref** — 第 2 张及以后传入前一张图路径作为参考：
   ```
   generate_image(
     prompt="...",
     outputPath="./output/topic-02.png",
     referenceImages=["./output/topic-01.png"]
   )
   ```
3. **保持一致**:
   - 配色方案不变
   - 字体风格不变
   - 装饰元素风格不变
   - 版式节奏可变（封面用 sparse → 内容用 balanced/dense → 结尾用 sparse）

### 系列图位置策略
| 位置 | 推荐版式 | 作用 |
|------|----------|------|
| 封面（第 1 张） | sparse | Hook，吸引注意力 |
| 内容（第 2-N-1 张） | balanced / dense / list | 传递核心价值 |
| 结尾（最后 1 张） | sparse | CTA，强化记忆 |
