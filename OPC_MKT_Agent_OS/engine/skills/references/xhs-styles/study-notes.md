# Study-Notes 学习笔记风格

## 配色方案
- 主色: #1E3A5F (蓝色笔) — 主要手写文字
- 辅助色: #1A1A1A (黑色笔) — 次级文字
- 背景色: #FFF8E7 (米黄纸张) — 底色
- 文字色: #1E3A5F / #1A1A1A — 双笔系统
- 点缀色: #FFFF00 (黄色荧光笔, 50%透明度), #CC0000 (红笔批注)

## 三笔系统
1. **蓝/黑笔**: 主要内容书写
2. **黄色荧光笔**: 关键词标记（半透明覆盖）
3. **红笔**: 重点强调、批注、纠正标记

## 视觉元素
- 书桌俯拍视角
- 横线/方格笔记纸纹理
- 手写文字铺满整页
- 便签条/胶带粘贴效果
- 边距注记（margin notes）
- 页码/日期标注
- 简单符号: → * ✓ ✗ ！

## 排版风格
- 标题: 手写粗体，红色或蓝色
- 正文: 潦草但可读的手写体，紧凑行距
- 强调: 荧光笔高亮、红笔圈注、箭头指向

## 内容结构
1. 核心主题（大标题）
2. 编号步骤/要点
3. 边距批注补充

## 推荐版式
- dense ★★★ — 极致内容密度
- list ★★★ — 编号笔记
- mindmap ★★★ — 知识关系图

## 适用场景
- 学习指南: 真实笔记感
- 考试笔记: 应试氛围
- 知识总结: 框架性内容
- 读书笔记: 阅读心得
- 教程速记: 速记风格

## 不适用场景
- 品牌形象类
- 需要精美设计的内容
- 情绪/氛围类

## 应该避免
- 复杂 emoji
- 过多留白
- 精美格式化
- 装饰性色彩
- 卡通元素
