# 视觉内容生成 Agent SOP

## 启动前必读
1. 读取 `./memory/context/brand-voice.md`（品牌调性）
2. 读取 `./memory/context/target-audience.md`（目标受众）
3. 确认内容类型（封面图/海报/配图/短视频脚本）

## 内容类型

| 类型 | 适用场景 | 产出物 | 关键要素 |
|------|---------|--------|----------|
| 封面图 | 小红书/公众号封面 | 图片生成 Prompt + 文字排版建议 | 3 秒内抓住注意力 |
| 海报 | 产品发布/活动推广 | 设计描述 + 文案 + 布局建议 | 信息层级清晰 |
| 社交配图 | 推文/笔记配图 | 图片 Prompt + 风格说明 | 与文案风格一致 |
| 短视频脚本 | 抖音/视频号/Reels | 分镜脚本 + 字幕 + BGM 建议 | 前 3 秒黄金 Hook |

## 图片生成规范

### Prompt 工程
- 使用英文撰写图片生成 Prompt（AI 模型对英文理解更好）
- 结构：[主体描述] + [风格] + [色调] + [构图] + [细节]
- 明确指定不要出现的元素（negative prompt）
- 每个 Prompt 同时给出简短版（DALL-E）和详细版（Midjourney/SD）

### 品牌视觉规范
- 色调：与品牌主色调一致（参考 brand-voice.md）
- 风格：简洁现代、科技感、留白克制
- 字体：无衬线体为主，中文用思源黑体/苹方
- 避免：过度装饰、低质量素材感、AI 味太重的光影

### 尺寸参考
| 平台 | 封面 | 配图 | 视频 |
|------|------|------|------|
| 小红书 | 1080x1440 (3:4) | 1080x1080 (1:1) | 1080x1920 (9:16) |
| X/Twitter | 1200x675 (16:9) | 1200x1200 (1:1) | 1920x1080 (16:9) |
| 公众号 | 900x383 (2.35:1) | 1080x720 (3:2) | - |

## 短视频脚本规范

### 结构
- 开头（0-3秒）: Hook — 文字/问题/冲突画面
- 正文（3-25秒）: 核心内容 — 分 3-5 个镜头
- 结尾（25-30秒）: CTA — 关注/点赞/评论引导

### 分镜格式
```
| 镜头 | 时长 | 画面描述 | 字幕/旁白 | 音效/BGM |
|------|------|----------|-----------|----------|
| 1    | 0-3s | [画面]   | [文字]    | [音效]   |
```
