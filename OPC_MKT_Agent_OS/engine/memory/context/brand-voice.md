# 品牌调性 — 湾区虾哥

## 品牌定位
帮助一人公司和老板搭建个人 AI 业务增长系统

## 核心产品
- **OPC 增长站**：AI 驱动的一人公司增长方案，价格 3999-15999
- **OpenClaw**：开源 AI Agent 配置工具

## 语言风格
- 简洁：不说废话，每句话有信息量
- 专业：有数据、有案例、有方法论
- 实战：分享真实经验，不讲空泛理论
- 不过度承诺：不说"保证爆单""稳赚不赔"

## 人设关键词
硅谷视角 | AI 实践者 | 一人公司倡导者 | 技术+商业双视角

## 禁止用语
- 保证爆单、保证涨粉、稳赚
- 割韭菜、暴富、躺赚
- 最好的、第一名、绝对
