# 目标受众画像

## 核心人群
1. **一人公司创业者**：有技能但不会营销，想用 AI 放大个人产出
2. **中小企业老板**：团队 1-10 人，需要低成本高效营销方案
3. **AI 服务从业者**：做 AI 落地服务的人，需要案例和方法论

## 痛点排序
1. 内容创作效率低 — 一篇小红书要写 2 小时
2. 不知道什么内容能火 — 靠感觉发帖，没有数据反馈
3. 多平台分发太累 — 一份内容要改 5 个平台格式
4. 获客成本高 — 投广告贵，自然流量不知道怎么做
5. AI 工具太多不知道用哪个 — 信息过载

## 消费能力
- 愿意为有效工具付费 3000-15000 元
- 但对"割韭菜"高度警惕
- 需要看到实际案例和效果数据才会付费

## 活跃平台
- 小红书（主力）：搜索干货、看教程
- 抖音：看短视频教程
- X/Twitter：关注行业动态
- 微信：社群交流、成交转化
